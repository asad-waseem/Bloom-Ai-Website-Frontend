# **App Name**: AI Bloom

## Core Features:

- Hero Section Animation: Animated background in the hero section to capture visitor attention.
- AI Capability Cards: Interactive cards showcasing AI capabilities with concise descriptions. Each card links to a detailed page or demo.
- Testimonials Carousel: Customer testimonials carousel to build trust and credibility. Displayed using a sleek, animated design.
- Contact Form: An integrated contact form for easy communication. Include fields for name, email, and message.
- Intelligent Contact Routing: Use AI to analyze the message content from the contact form and route it to the appropriate department using a tool.

## Style Guidelines:

- Background color: Dark gray (#121212) to provide contrast and depth, highlighting the vibrant accent colors.
- Primary color: Vibrant purple (#9F57FF) for headlines and key elements, conveying innovation and creativity.
- Accent color: Electric pink (#FF47D3) for calls to action and highlights, adding energy and a modern touch.
- Body and headline font: 'Inter', a sans-serif font, combining modern aesthetic with a neutral feel.
- Use minimalist, custom-designed icons in line with the color scheme. Icons should represent AI concepts and platform features.
- Modern, clean layout with a focus on user experience. Use a grid-based design with ample white space to improve readability and visual appeal.
- Subtle, dynamic animations on the hero section background. Micro-interactions on the interactive cards and a smooth transition effect for the testimonial carousel.
