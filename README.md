# AI Bloom | The Intelligent AI Platform Showcase

This is a high-performance, Next.js 15 based AI platform showcase. Engineered for "million-dollar" aesthetics with 60+ FPS smoothness and Apple-inspired UI.

## 🚀 Getting Started

### 1. Prerequisites
- Node.js 18.x or higher
- NPM or PNPM

### 2. Installation
```bash
npm install
```

### 3. Environment Setup
Create a `.env` file in the root directory and add your Gemini API Key:
```env
GEMINI_API_KEY=your_api_key_here
```

### 4. Run Development Server
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 💎 Features
- **Ultra-Smooth UI**: Optimized Canvas particles and CSS rendering hints for 60+ FPS.
- **Apple Glassmorphism**: High-end frosted glass effects and refined typography.
- **AI Integration**: Built-in Genkit flows for Chatbots and Intelligent Contact Routing.
- **WhatsApp Bridge**: Direct live-chat connection for enterprise support.
- **Responsive Design**: Fully optimized for Mobile, Tablet, and Desktop.

## 🛠 Tech Stack
- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS + Shadcn/UI
- **AI**: Google Genkit + Gemini 2.0
- **Animations**: Framer Motion / Lucide / Custom Canvas Engine

## 📤 Deployment
This project is ready for deployment on **Vercel** or **Firebase App Hosting**. 
Ensure you add your `GEMINI_API_KEY` to the production environment variables.
