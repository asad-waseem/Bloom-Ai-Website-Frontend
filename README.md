# AI Bloom | The Intelligent AI Platform Showcase

This is a high-performance, Next.js 15 based AI platform showcase. Engineered for "million-dollar" aesthetics with 60+ FPS smoothness and Apple-inspired UI.

## 🚀 Getting Started Locally

### 1. Prerequisites
- **Node.js**: 18.x or higher
- **NPM** or **PNPM**
- **Git**: Installed and configured

### 2. Initial Setup
Once you have downloaded the ZIP file and extracted it:
```bash
# Navigate to the project folder
cd ai-bloom

# Install dependencies
npm install
```

### 3. Environment Configuration
Create a `.env` file in the root directory and add your Gemini API Key:
```env
GEMINI_API_KEY=your_actual_gemini_api_key_here
```
*Note: You can get a key from the [Google AI Studio](https://aistudio.google.com/).*

### 4. Run Development Server
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) in your browser.

## 📤 Deploying to GitHub

Follow these steps to push your code to a new GitHub repository:

1. **Create Repository**: Go to GitHub and create a new, empty repository named `ai-bloom`.
2. **Initialize Git**:
   ```bash
   git init
   git add .
   git commit -m "Initial commit: Million-dollar AI Platform"
   ```
3. **Connect & Push**:
   ```bash
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/ai-bloom.git
   git push -u origin main
   ```

## 🛠 Tech Stack Details
- **Frontend**: Next.js 15 (App Router), React 19, Tailwind CSS.
